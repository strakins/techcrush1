# Testimonials Grid Section

A responsive testimonial grid layout built with **HTML** and **Tailwind CSS**, 

---

---

## 🚀 Features

[View the Project here](https://strakins.github.io/techcrush1/)

* ✅ Fully responsive (Mobile, Tablet, Desktop)
* ✅ CSS Grid-based layout
* ✅ Tailwind CSS custom theme (colors & typography)
* ✅ Quote SVG background styling
* ✅ Responsive card reordering (mobile-first UX)
* ✅ Clean and reusable structure

---

## 🛠️ Tech Stack

* **HTML5**
* **Tailwind CSS (CDN)**
* **Google Fonts (Barlow Semi Condensed)**

## ▶️ Getting Started

1. Clone or download the project
2. Ensure all assets are in the `/assets` folder
3. Open `index.html` in your browser

No build tools required — Tailwind is loaded via CDN.
